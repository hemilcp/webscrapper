Technology/Tools used:

Java 8
Selenium 2.0
AngularJS
MongoDB 3.5
Git
Eclipse
Tomcat 9


To run the project:
1) Import project into Eclipse. (Make sure you have JDK 1.8.* installed)
2) Install MongoDB for the database.
2) As we are using Selenium to automate the browser testing, download geokodriver (or chromedrive, if you are using chrome. In this project we are using Mozilla firefox as our browser) and replace the driver path in properties file. (located under resources)
3) maven build the project.
4) Run the tomcat server and test.
